# FiveM-TextColors
This is simple guide for Fivem Text Colors
```lua
Can use script text:
~r~ = Red
~b~ = Blue
~g~ = Green
~y~ = Yellow
~p~ = Purple
~c~ = Grey
~m~ = Dark Grey
~u~ = Black
~o~ = Orange

Special:
~n~ = New line
~s~ = Reset Color
~h~ = Bold text

For notification:
∑ = Rockstar Icon
÷ = Rockstar Icon
¦ = Rockstar Verified Icon

Can use to server title:
^1 = red
^2 = green
^3 = yellow
^4 = blue
^5 = CYAN
^6 = Pink
^7 = white
^8 is a color that changes depending what level you are on.
^9 = gray
^0= black
^c = Blandishbeige

^0 is based on the RGB of the message sent
```
